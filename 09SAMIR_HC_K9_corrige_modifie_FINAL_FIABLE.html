SYNCHRONISATION EN LIGNE (TEMPS RÉEL)
Collaboration en ligne : tous les collègues se connectent au même Room.
(Choisis: Supabase بدون سيرفر ✅ أو Socket.IO أو Firebase)
Supabase: تحتاج Project URL + anon key فقط. لا تستعمل secret key.
Socket.IO: تحتاج رابط سيرفر Node.js فيه socket.io.
Firebase: تحتاج firebaseConfig JSON + تفعيل Anonymous Auth + قواعد Rules.
المسار: Supabase → Project Settings → API → (Project URL + anon/public key)
المسار: Firebase Console → Project settings → General → Your apps (Web) → firebaseConfig
⚠️ استعمل Realtime Database (databaseURL) وفعّل Anonymous Auth.
ننصح بقواعد Rules: القراءة للجميع (auth) والكتابة للمسؤولين فقط (admins).
✅ Reset يمسح بيانات هذا الجهاز فقط (localStorage + IndexedDB + Cache) ثم يعيد تشغيل الصفحة. لا يمسح Firebase عند الزملاء.
